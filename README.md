# 🧠 Hospital General Information Analysis using Python

## 📌 Project Overview
This project focuses on cleaning, preprocessing, and analyzing hospital general information data using Python. The project includes data cleaning, exploratory data analysis (EDA), statistical analysis, feature engineering, and data visualization to identify useful insights.

## 🎯 Project Objectives
- Clean and preprocess the dataset
- Handle missing values and duplicates
- Perform Exploratory Data Analysis (EDA)
- Create meaningful visualizations
- Generate business insights

## 📂 Dataset
- Dataset: Hospital General Information
- Format: CSV
- Rows: 2000+
- Columns: 15+

## 🛠️ Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📊 Project Workflow
1. Import Libraries
2. Load Dataset
3. Data Cleaning
4. Data Preprocessing
5. Exploratory Data Analysis
6. Data Visualization
7. Insights & Conclusion

## 📈 Key Visualizations
- Bar Chart
- Line Chart
- Pie Chart
- Histogram
- Box Plot
- Heatmap

## 💡 Key Insights
- Most hospitals are located in urban areas.
- Missing values were handled successfully.
- Performance varies across hospital types.
- Visualizations helped identify important trends.

## ✅ Conclusion
This project demonstrates how Python can be used to clean, analyze, and visualize healthcare data for better decision-making.
